1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programs to manipulate the document's content, structure, and style.

2. How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById('id'), document.querySelector('.class'), or document.getElementsByTagName('tagName'). These methods return the first matching element, or a collection of elements, allowing you to manipulate them further.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique that allows you to attach a single event listener to a parent element instead of attaching listeners to individual child elements. This is useful because it improves performance, especially when dealing with many child elements, and allows dynamically added elements to inherit the event listener without needing to add listeners to them separately.

4. How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript. For attributes, you can use element.setAttribute('attributeName', 'value') or directly access attributes like element.src. For styles, you can use element.style.propertyName = 'value' to set CSS styles directly, or element.classList.add('className') to add a class for styling.
